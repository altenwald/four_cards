    How to Play

    If you want to make suggestions or obtain the source code you can go to Github site and create a issue, fork the code to send us modifications or fixes or whatever.

    Starting with a Match

    You have to provide a screen name for you and then you are going to be redirected to the waiting screen. This screen will show you the players incoming to play with you, the URL and QR code to join to the match and the possibility to add bots to your match.

    CAUTION: do NOT press the Deal button until everyone is inside of the game. Deal means you are going to start the game and it's not possible to go back:

    Once the game is over, you will get two options:

    The "New One" button let us to start a completely new game which is letting to us to reconfigure the people who is going to play with us.

    The "Restart" is going into the new match with exactly the same people to play with.

    Playing to Four Cards

    Four Cards is a game for 2-4 players. The mission of the game is being the first one dropping all of their cards. You start with 4 cards and you can only play during your turn and drop one card based on the following rules.

    Rules

    • The card to be dropped on the table should match in number to the card "shown". For example:

      This card is the 2 of swords. So, the card you can play should be 2 of gold, 2 of cups or 2 of clubs.
    • If you have no cards to play, you can pass:
    • The game is over when you drop the last card you had. Who has more captured cards wins!

    Kiosk

    If you want to play with others in the same room, the best configuration is playing using your phone mobiles and putting the Kiosk screen in a computer or in the TV. You can load a new match using this URL:

    https://fourcards.altenwald.com/kiosk.

    The Kiosk only shows the information of the shown card, the table of the turns, the log view and when the match is over, the information about the high scores for the rounds.
